Merge base links
================

Introduction
------------

This processor merge all base links to one.

``config.json`` entries
-----------------------

.. code-block:: javascript

    {
        // ...
        // General import options (see config.json documentation)
        // ...

        // Merge all base links to one
        "merge_base_links": true
    }

``merge_base_links`` *(default: false)*
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If set to ``true``, all base links will be merge in a root lik.
